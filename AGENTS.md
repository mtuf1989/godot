# Symphony Project Agents

## Required Skills

The following skills MUST always be activated for this project:

- `godot-gdscript` — For typed GDScript implementation, signal connections, lifecycle callbacks, and Godot 4 scripting conventions.
- `godot-gdextension-cpp` — For GDExtension C++ code, godot-cpp setup, ClassDB registration, GDCLASS macro, ownership discipline, and native bindings.

## Project Context

This project (Symphony) is a C++ module for Godot Engine implementing a real-time node-based DSP flow graph. It requires both native C++ development (the module itself) and GDScript knowledge (for the API surface, test scenes, and integration).

Always apply both skills when working on any task in this repository.
